AEM 6.x add-on which extends Slice functionality to support AEM version 6.0, 6.1 and 6.2. It defines additional Guice modules which provides AEM-specific objects like Page, PageManager, etc.

AEM 6.x ADD-ON

Maven dependencies

Add the following dependencies to your project

<dependency>
    <groupId>com.cognifide.slice</groupId>
    <artifactId>slice-core-api</artifactId>
    <version>4.3.1</version>
</dependency>
<dependency>
    <groupId>com.cognifide.slice</groupId>
    <artifactId>slice-core</artifactId>
    <version>4.3.1</version>
</dependency>
<dependency>
    <groupId>com.cognifide.slice</groupId>
    <artifactId>slice-mapper</artifactId>
    <version>4.3.1</version>
</dependency>
<dependency>
    <groupId>com.cognifide.slice</groupId>
    <artifactId>slice-mapper-api</artifactId>
    <version>4.3.1</version>
</dependency>

 

If you use Slice in AEM/CQ add also the add-on dependency:

AEM 6.x

<dependency>
    <groupId>com.cognifide.slice-addon</groupId>
    <artifactId>slice-aem60</artifactId>
    <version>1.2.0</version>
</dependency>

CQ 5.6

<dependency>
    <groupId>com.cognifide.slice-addon</groupId>
    <artifactId>slice-cq56</artifactId>
    <version>2.1.0</version>
</dependency>
